#🕵️ Sherlock Holmes Quiz

A simple interactive web quiz about the world of Sherlock Holmes, created using HTML, CSS, and JavaScript.

The quiz tests your knowledge of Sherlock Holmes stories written by Arthur Conan Doyle and lets you discover your detective level at the end.

📖 Description

This project is a browser-based quiz where users answer multiple-choice questions related to the adventures of Sherlock Holmes.

The quiz includes:

Multiple Sherlock Holmes trivia questions

A progress bar showing quiz completion

Visual feedback for correct and incorrect answers

A final detective level based on the score

Background mystery music

A rating system at the end of the quiz

The design uses themed images to create a Victorian detective atmosphere.

🧩 Features

📊 Progress Bar showing quiz progress

🧠 Multiple Choice Questions

✅ Correct Answer Highlight

❌ Wrong Answer Highlight

🕵️ Detective Level Result

🎮 How the Quiz Works

1.The quiz loads the first question.
2.The user selects one answer.
3.When pressing Next, the quiz:
  Shows the correct answer
  Highlights incorrect selections
4.After the final question, the user receives a Detective Level based on their score.

🧠 Detective Levels

The quiz evaluates your knowledge of Sherlock Holmes:

Score	Level
0–3	Beginner Detective
4–6	Apprentice Detective
7–9	Skilled Detective
10+	Master Detective
